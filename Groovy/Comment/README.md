## One line comment
```groovy
class MyFirstGroovyProgram {
    static void main(args) {

        //single line command
   
        def age = 40
        println(age)

    }
}
```

## Multiple lines comment
```groovy
class MyFirstGroovyProgram {
    static void main(args) {

        //single line command

        /*
        this is 
        a multi-line 
        command
        */
        
        def age = 40
        println(age)

    }
}
```
