## Data Types In Groovy
- int
- float
- Boolean
- String

## Example 1
```groovy
String s = 'Tom'
int i = 4654
float f = 38.27
boolean flag = true

println(s)
println(i)
println(f)
println(flag)

println("$s, $i, $f, $flag")
```

**OUTPUT:**
```
Tom
4654
38.27
true
Tom, 4654, 38.27, true
```
